# Como usar:

**1 - Abra uma nova aba no terminal**
**2 - Execute o seguinte comando**
  ```
  yarn run json-server -w -p 3333 ./mock/database.json
  ```
  **Ou**
  ```
  npm run json-server -w -p 3333 ./mock/database.json
  ```
  **Ou se preferir**
   ```
  yarn mock
  ```
  **Ou**
  ```
  npm run mock
  ```
